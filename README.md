📋 نظرة عامة

Pulse هو نظام طبي ذكي متكامل مصمم خصيصاً للأطباء والعيادات الطبية لإدارة سجلات المرضى، تحليل التحاليل الطبية، وتقديم توصيات ذكية مخصصة بناءً على نتائج التحاليل. النظام يوفر واجهة مستخدم سهلة الاستخدام تدعم اللغتين العربية والإنجليزية.

🚀 التثبيت

المتطلبات الأساسية

· متصفح ويب حديث (Chrome 90+, Firefox 88+, Safari 14+)
· تفعيل JavaScript
· مساحة تخزين (50MB كحد أدنى)

خطوات التثبيت

1. تحميل المشروع
   ```bash
   git clone https://github.com/your-username/pulse-medical-system.git
   cd pulse-medical-system
   ```
2. تشغيل النظام
   ```bash
   # باستخدام Python
   python -m http.server 8000
   
   # أو باستخدام Node.js
   npx http-server -p 8000
   
   # أو باستخدام PHP
   php -S localhost:8000
   ```
3. الوصول للنظام
   ```
   http://localhost:8000
   ```

📖 الاستخدام

🔐 التسجيل والدخول

1. إنشاء حساب جديد
   · انتقل إلى register.html
   · املأ البيانات المطلوبة (الاسم، البريد، التخصص، etc.)
   · أنشئ كلمة مرور قوية
   · اضغط على "إنشاء الحساب"
2. تسجيل الدخول
   · انتقل إلى login.html
   · أدخل البريد الإلكتروني وكلمة المرور
   · اختر "تذكرني" إذا رغبت
   · اضغط على "تسجيل الدخول"

👥 إدارة المرضى

1. إضافة مريض جديد
   ```javascript
   // مثال: إضافة مريض برمجياً
   const patientData = {
     name: "محمد أحمد",
     medicalId: "MED123456",
     gender: "ذكر",
     birthDate: "1985-05-15",
     bloodType: "A+"
   };
   ```
2. بحث وتصفية
   · استخدم شريط البحث للبحث بالاسم أو الرقم الطبي
   · استخدم الفلاتر للتصفية حسب الجنس أو التاريخ

🔬 تحليل التحاليل

1. إضافة تحليل جديد
   · اختر المريض من القائمة
   · اختر نوع التحليل (سكر الدم، وظائف الكبد، etc.)
   · أدخل القيم المخبرية
   · اضغط على "تحليل النتائج"
2. قراءة النتائج
   · عرض النتائج مع تصنيف (طبيعي، تحذير، حرج)
   · التوصيات المخصصة حسب الحالة
   · الرسوم البيانية التوضيحية

📊 إنشاء التقارير

1. تقرير المريض
   · تقرير مفصل عن حالة المريض
   · سجل التحاليل والتوصيات
   · إمكانية الطباعة والتصدير
2. تقارير إحصائية
   · إحصائيات عامة عن العيادة
   · نسب النتائج وتحليل الأداء
   · رسوم بيانية تفاعلية
